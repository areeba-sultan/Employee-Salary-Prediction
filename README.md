# Employee Salary Prediction System

## Project Overview

This project predicts employee salaries using an Artificial Neural Network (ANN) model. The system analyzes employee-related features such as experience, education, and other relevant factors to estimate salary accurately.

## Features

* Employee salary prediction using Machine Learning
* Data preprocessing and cleaning
* Feature scaling and normalization
* Artificial Neural Network (ANN) model
* Model training and testing
* Performance evaluation

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn
* TensorFlow / Keras (if used)

## Installation

Install the required libraries:

pip install numpy pandas scikit-learn

## How to Run

Run the Python file:

python project.py

## Output

The model predicts employee salaries based on the provided input features and displays the predicted salary.
